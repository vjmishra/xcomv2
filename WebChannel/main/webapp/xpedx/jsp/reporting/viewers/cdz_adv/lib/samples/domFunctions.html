DhtmlLib Definition
The DhtmlLib is a JavaScript-based library. It provides a set of widgets to build nicely and user friendly UI in web application.
The DHTML Library provides basic component like text field, label and list box, and also rich component like tree view, tab and toolbar.
The UI of WebI Interactive Viewer for instance is completely built with the DTML Library.
Some samples on how to start using DHTML Library widgets are available after adding DHTML Library into your web application.
